# Abid Portfolio

A personal portfolio website showcasing frontend development skills, built using **HTML**, **Tailwind CSS**, and **AOS (Animate On Scroll)** for smooth scroll animations.

## Features

- Clean and modern design
- Responsive layout optimized for all devices
- Scroll animations powered by AOS library
- Built with Tailwind CSS for utility-first styling
- Easy to customize and extend

## Technologies Used

- HTML5
- Tailwind CSS
- AOS (Animate On Scroll) for animation effects

## Live Demo

Check the live site at [abid-folio.vercel.app](https://abid-folio.vercel.app)

## Getting Started

### Prerequisites

- Modern web browser
- Optional: Node.js and npm if you want to install dependencies and run a local development server

### Installation

1. Clone the repository:
```git clone https://github.com/00Abid/abid-folio.git```


2. Open the project folder and open `index.html` in your browser.

### Development

If using Tailwind CSS via npm and building CSS:

1. Install dependencies:
```npm install```

2. Run build/watch commands as per your setup.
